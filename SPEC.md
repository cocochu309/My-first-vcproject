# 汉字学习工具 - 规格文档

## 1. 项目概述

- **项目名称**: 萌趣学汉字
- **项目类型**: 教育类Web应用 (单HTML文件)
- **核心功能**: 为5-6岁儿童提供汉字学习体验，包含文字、拼音、组词、造句学习，以及笔画动画演示
- **目标用户**: 5-6岁学前儿童（幼小衔接）

## 2. UI/UX 规格

### 布局结构

- **整体布局**: 全屏单页应用，分区明确
- **顶部区域**: 标题栏 + 分组导航
- **中部区域**: 学习卡片（汉字展示 + 信息标签页）
- **底部区域**: 操作按钮

### 响应式断点
- 移动端: < 768px
- 平板: 768px - 1024px
- 桌面: > 1024px

### 视觉设计

#### 色彩方案
- **主色**: #FF6B6B (珊瑚红 - 活泼友好)
- **辅色**: #4ECDC4 (薄荷绿 - 清新舒适)
- **强调色**: #FFE66D (阳光黄 - 温暖鼓励)
- **背景色**: #FFF9E6 (米黄色 - 护眼)
- **文字色**: #2C3E50 (深灰蓝 - 清晰易读)
- **卡片背景**: #FFFFFF

#### 字体
- **汉字展示**: "KaiTi", "STKaiti", "楷体", serif (楷体更适合儿童)
- **拼音**: "Comic Sans MS", cursive
- **其他文字**: "Noto Sans SC", sans-serif
- **字号**:
  - 汉字: 120px (卡片), 48px (列表)
  - 拼音: 24px
  - 组词/造句: 18px
  - 按钮: 16px

#### 间距系统
- 基础单位: 8px
- 卡片内边距: 24px
- 元素间距: 16px

### 组件规格

#### 1. 分组导航栏
- 横向滚动展示100个分组
- 每组20个汉字（100组 × 20字 = 2000字）
- 当前选中分组高亮显示
- 分组按钮样式: 圆角矩形，悬停放大效果

#### 2. 学习卡片
- 大号汉字展示区域（带笔画动画）
- 拼音显示（带声调）
- 信息标签页：文字 | 组词 | 造句
- 卡片尺寸: 400px × 500px (桌面), 自适应移动端

#### 3. 导航控件
- 左侧/右侧箭头切换汉字
- 底部圆点指示当前位置
- 支持键盘左右键操作

#### 4. 掌握按钮
- 大尺寸圆形按钮
- 点击后触发点赞鼓励动画
- 按钮颜色: #4ECDC4 (薄荷绿)

#### 5. 点赞鼓励动画
- 动画类型: 烟花绽放 + 星星飞散 + "你真棒！"文字
- 动画时长: 2秒
- 颜色: 多彩渐变

## 3. 功能规格

### 3.1 汉字浏览
- 按分组查看汉字（100组，每组20字）
- 支持左右切换查看
- 显示当前进度（如：3/20）

### 3.2 汉字信息展示
- **文字标签**: 显示汉字大字，带笔画动画
- **拼音标签**: 显示带声调的完整拼音
- **组词标签**: 显示3-5个常用词汇
- **造句标签**: 显示1-2个简单例句

### 3.3 笔画动画
- 使用SVG路径绘制笔画
- 逐笔显示动画效果
- 可重播动画
- 笔画顺序正确

### 3.4 掌握功能
- 点击"掌握啦"按钮
- 弹出鼓励动画
- 记录已掌握状态（本地存储）
- 显示掌握进度

### 3.5 进度追踪
- 本地存储已掌握汉字
- 显示总体掌握进度百分比
- 分组进度可视化

## 4. 词汇数据

### 数据结构
```javascript
{
  "1": [  // 分组1
    {
      "char": "一",
      "pinyin": "yī",
      "words": ["一个", "第一", "一天"],
      "sentence": "我有一个苹果。"
    },
    // ... 20个汉字
  ],
  "2": [  // 分组2
    // ...
  ]
  // ... 100个分组
}
```

### 词汇来源
- 选取小学一年级常用汉字
- 优先选择笔画简单、组词常用的字
- 确保覆盖基础词汇
